🌐 Cara Menjalankan via HTTP

Firebase membutuhkan akses lewat http:// atau https://, bukan file://. Pilih salah satu cara berikut:

A

Python (sudah terinstall di hampir semua komputer)

Buka Terminal / Command Prompt, masuk ke folder tempat file berada, lalu jalankan:

cd /path/ke/folder/file
python -m http.server 8080

Lalu buka browser dan akses: http://localhost:8080/exam-app.html
Bagikan alamat ini ke peserta di jaringan yang sama: http://[IP-komputer-Anda]:8080/exam-app.html

B

VS Code + Live Server (untuk pengguna VS Code)

Install ekstensi Live Server dari Ritwick Dey → klik kanan file exam-app.html → pilih "Open with Live Server". Browser akan terbuka otomatis.

C

Netlify Drop (hosting gratis, paling mudah untuk peserta jarak jauh)

Buka app.netlify.com/drop → drag & drop file exam-app.html ke halaman tersebut → Netlify akan memberi URL publik yang bisa diakses dari mana saja, misalnya https://amazing-name-123.netlify.app.

✓

Rekomendasi untuk Lab Komputer

Jalankan Python server di komputer admin/pengawas, lalu peserta akses lewat IP komputer admin di jaringan LAN yang sama. Semua peserta pakai satu URL yang sama — tidak perlu copy file ke tiap komputer.

☁️ Koneksi Cloud (Firebase)

Hubungkan aplikasi ke Firebase Firestore agar hasil ujian peserta langsung tersimpan di cloud dan bisa Anda akses dari perangkat mana pun.

1

Buka Firebase Console

Buka console.firebase.google.com → Login dengan akun Google → Klik "Add project" → beri nama (misal: SecureExam-Sekolah) → klik Continue hingga selesai.

2

Aktifkan Firestore Database

Di menu kiri pilih Build → Firestore Database → klik "Create database" → pilih "Start in test mode" → pilih region terdekat (misal: asia-southeast1) → klik Enable.

3

Daftarkan Aplikasi Web

Di Project Overview → klik ikon </> (Web) → beri nama app → klik "Register app". Firebase akan menampilkan blok kode berisi firebaseConfig. Salin seluruh isi objek { apiKey: ..., projectId: ..., ... } lalu tempel di kolom di bawah ini.

4

Atur Security Rules (Penting!)

Di Firestore → tab Rules → ganti isi rules dengan kode berikut lalu klik Publish:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // Soal ujian (tanpa kunci jawaban) — peserta boleh baca
    match /se_exams/{doc} {
      allow read:  if true;
      allow write: if request.auth != null;
    }

    // Kunci jawaban — hanya admin (login) yang boleh baca & tulis
    match /se_answers/{doc} {
      allow read:  if request.auth != null;
      allow write: if request.auth != null;
    }

    // Jawaban peserta — peserta boleh create, admin boleh baca
    match /se_submissions/{doc} {
      allow create: if true;
      allow read:   if request.auth != null;
      allow update: if request.auth != null;
      allow delete: if false;
    }

    // Hasil ujian — admin tulis, semua boleh baca
    match /se_results/{doc} {
      allow read:   if true;
      allow write:  if request.auth != null;
    }

    // Percobaan ujian — semua perangkat
    match /se_attempts/{doc} {
      allow read, write: if true;
    }
  }
}
